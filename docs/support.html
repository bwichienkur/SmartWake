Alarms not ringing?

  1. Allow notifications for SmartWake
  2. Disable battery optimization (Android: Settings → Apps → SmartWake → Battery → Unrestricted)
  3. Use Settings → Test Alarm (1 min) to verify notifications
  4. Ensure Do Not Disturb allows alarms / time-sensitive notifications

Smart Wake accuracy

Connect Apple Health or a wearable for better estimates. Complete the 7-day calibration week. Sleep stages are estimates only — not medical-grade.

Restore Premium

Open SmartWake → Settings → Restore Purchases. Use the same Apple ID or Google account used to subscribe.
